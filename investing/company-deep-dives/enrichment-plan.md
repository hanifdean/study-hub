## Enrichment Sections to Add (Per Page)

### 1. Primary Sources Card
- Link to FY2024 annual report PDF
- Link to latest quarterly financial statements
- Link to investor presentation
- Link to IDX filing page

### 2. Valuation Framework
- Current P/E, P/BV, EV/EBITDA (with source date)
- 5-year historical range for key multiples
- Peer comparison table (domestic + regional peers)
- Fair value estimate methodology

### 3. Investment Thesis (Structured)
- Bull case: core thesis + key assumptions + catalysts + upside target
- Base case: expected outcome with moderate assumptions
- Bear case: what breaks the thesis + downside target
- Probability-weighted expected value

### 4. Thesis Killers (Explicit)
- Specific events/metrics that invalidate the bull case
- Red flags to monitor
- Historical precedents of similar thesis failures
- "Get out if..." triggers

### 5. Operational KPIs Dashboard
- Company-specific metrics (not generic financials)
- Quarterly tracking table
- Benchmark comparisons

### 6. Capital Allocation & Governance
- Shareholder structure + free float
- ROE/ROIC trends
- Debt maturity profile
- Management track record scorecard
- Related party transaction history

### 7. Scenario Analysis Table
- 3 scenarios with assumptions, revenue/profit impact, target price
- Probability weights
- Expected value calculation
